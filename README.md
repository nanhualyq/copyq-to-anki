# Install
1. copyq
2. nodejs
3. anki + plugin(anki-connect)

# Run
```
npm install
npm run build
node dist/index.js
```

Suggested one-click trigger via global shortcut, you can register a global shortcut key for this script through copyq or operating system settings.
